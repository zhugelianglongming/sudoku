## 规则

| 序号  | 限制区域 | 限制规则               | 备注  |
|:---:|:----:|:-------------------|:---:|
|  1  |  行   | [1~9填充]            |     |
|  2  |  列   | [1~9填充]            |     |
|  3  |  宫   | [1~9填充]            |     |
|  4  |  全盘  | 对于任意格，其[马步]格 ≠ 当前格 | 无标  |

> 规则与 [养马数独] 相同（即无标的 [养马数独]）
