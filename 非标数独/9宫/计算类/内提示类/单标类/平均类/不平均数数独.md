# 不平均数数独

## 规则

| 序号  | 限制区域 | 限制规则             | 备注  |
|:---:|:----:|:-----------------|:---:|
|  1  |  行   | [1~9填充]          |     |
|  2  |  列   | [1~9填充]          |     |
|  3  |  宫   | [1~9填充]          |     |
|  4  |  全盘  | 任意格 ≠ 左右两格的算数平均数 | 无标  |
|  5  |  全盘  | 任意格 ≠ 上下两格的算数平均数 | 无标  |

> 规则与 [平均数数独] 相同（即无标的 [平均数数独]）
