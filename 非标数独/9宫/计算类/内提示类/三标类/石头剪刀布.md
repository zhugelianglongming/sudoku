## 规则

| 序号 | 限制区域 | 限制规则 | 备注 |
| :---: | :---: | :--- | :---: |
| 1 | 行 | [1~9填充] | |
| 2 | 列 | [1~9填充] | |
| 3 | 宫 | [1~9填充] | |
| 4 | 标记顶点 | `X`：标记顶点周围四格<br/>- 存在对角的数字相同 | 全标 |
| 5 | 标记顶点 | 黑点：标记顶点周围四格<br/>- 不存在对角的数字相同<br/>- 存在对角数字是两倍关系 | 全标 |
