# 规则
| 序号  |  限制区域  | 限制规则                     |        备注        |
|:---:|:------:|:-------------------------|:----------------:|
|  1  |   行    | [1~9填充]                 |                  |
|  2  |   列    | [1~9填充]                 |                  |
|  3  |   宫    | [1~9填充]                 |                  |
|  4  | 提示数（边） | 提示数`8`两侧的[共边邻格]的积的个位数为 8 | 全标，即A * B = `*8` |
|  5  | 提示数（边） | 标记边`3`两侧的[共边邻格]的积的十位数为 3 | 全标，即A * B = `3*` |
